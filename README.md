# MMLECS-Emergency-Communication.
Morse Code Multi-Level Emergency Communication System using Arduino UNO
MMLECS: Morse Code Emergency System📝
Overview 
A portable embedded system for emergency communication using Morse Code. 
It features two modes: Manual Morse and a 4-level pre-loaded messaging system.  
🚀 Key Features* 4 Priority Levels: Critical, Urgent, Alert, and Info.
* Hardware NOT-Gate Logic: For inverted LED visual feedback.
* Bluetooth Integration: Wireless output via HC-05 module.
* Capacitive Touch Interface: Using TTP223 for intuitive input.  🛠️ Hardware Used* Arduino UNO.
* 74LS04 NOT Gate.
* TTP223 Touch Sensor.
